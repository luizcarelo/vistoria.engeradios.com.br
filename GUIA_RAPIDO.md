# Guia Rápido - Sistema Vistoria Remota ENGERADIOS

## 🚀 Primeiros Passos

### 1. Acesso ao Sistema

Acesse o sistema através do navegador:
- **URL:** http://seu-dominio.com.br/vistoria
- **Ou:** http://seu-ip/vistoria

### 2. Login Inicial

Use as credenciais de administrador:
- **E-mail:** admin@engeradios.com.br
- **Senha:** admin123

⚠️ **IMPORTANTE:** Altere a senha após o primeiro acesso!

## 👨‍💼 Fluxo de Trabalho do Administrador

### Passo 1: Cadastrar Supervisores

1. Faça login como administrador
2. No dashboard, clique em **"Gerenciar Supervisores"**
3. Preencha o formulário:
   - Nome completo do supervisor
   - E-mail (será usado para login)
   - Senha inicial
4. Clique em **"Cadastrar Supervisor"**
5. Anote as credenciais e repasse ao supervisor

### Passo 2: Cadastrar Clientes

1. No dashboard, clique em **"Gerenciar Clientes"**
2. Preencha o formulário:
   - Nome do cliente (obrigatório)
   - Endereço
   - Telefone
   - E-mail
3. Clique em **"Cadastrar Cliente"**

### Passo 3: Visualizar Vistorias

1. No dashboard, clique em **"Visualizar Vistorias"**
2. Use os filtros para buscar:
   - Por período (data início e fim)
   - Por cliente específico
   - Por supervisor específico
3. Clique em qualquer vistoria para ver os detalhes completos

## 👷 Fluxo de Trabalho do Supervisor

### Passo 1: Acessar o Sistema no Celular

1. Abra o navegador do celular (Chrome ou Safari)
2. Acesse a URL do sistema
3. Faça login com suas credenciais
4. **Opcional:** Adicione o app à tela inicial para acesso rápido

### Passo 2: Criar Nova Vistoria

1. No dashboard, clique em **"Nova Vistoria"**
2. Selecione o **cliente** que está sendo vistoriado
3. Preencha o **laudo da vistoria**:
   - Descreva detalhadamente o que foi encontrado
   - Relate problemas, não conformidades, etc.
4. Preencha o **orçamento de adequação**:
   - Liste materiais necessários
   - Descreva serviços a serem realizados
   - Inclua quantidades e especificações

### Passo 3: Adicionar Fotos

1. Clique em **"Adicionar Fotos"**
2. Tire fotos diretamente ou selecione da galeria
3. Você pode adicionar quantas fotos forem necessárias
4. Para remover uma foto, clique no **X** sobre ela

### Passo 4: Adicionar Áudio (Opcional)

1. Clique em **"Adicionar Áudio"**
2. Grave um áudio ou selecione um arquivo
3. Limite: 3 minutos
4. O áudio complementa o relatório escrito

### Passo 5: Concluir Vistoria

1. Revise todas as informações
2. Clique em **"Concluir e Enviar Vistoria"**
3. O sistema irá:
   - Salvar a vistoria no banco de dados
   - Enviar e-mail automático para operacional@engeradios.com.br
   - Confirmar o envio

### Passo 6: Consultar Vistorias Anteriores

1. No dashboard, clique em **"Consultar Vistorias"**
2. Use os filtros:
   - Data início e fim
   - Cliente específico
3. Clique em qualquer vistoria para ver detalhes
4. Você pode visualizar:
   - Laudo completo
   - Orçamento de adequação
   - Todas as fotos (clique para ampliar)
   - Áudio (se houver)

## 📱 Dicas para Uso no Celular

### Tirar Fotos de Qualidade

- Use boa iluminação
- Mantenha o celular estável
- Tire fotos de diferentes ângulos
- Capture detalhes importantes
- Evite fotos tremidas ou desfocadas

### Gravar Áudio Eficiente

- Grave em ambiente silencioso
- Fale de forma clara e pausada
- Mencione o cliente e data
- Descreva o que as fotos mostram
- Máximo de 3 minutos

### Economizar Dados Móveis

- Use Wi-Fi quando disponível
- Comprima fotos antes do upload (o sistema faz automaticamente)
- Evite fazer upload de vídeos

## 🔧 Solução de Problemas Comuns

### Não consigo fazer login

- Verifique se o e-mail está correto
- Verifique se a senha está correta (maiúsculas/minúsculas)
- Entre em contato com o administrador para resetar a senha

### Erro ao enviar fotos

- Verifique sua conexão com a internet
- Tente reduzir o número de fotos por vez
- Certifique-se de que as fotos não estão corrompidas

### E-mail não foi recebido

- Verifique a caixa de spam
- Confirme se a vistoria foi concluída com sucesso
- Entre em contato com o administrador

### Sistema lento no celular

- Feche outros aplicativos
- Limpe o cache do navegador
- Verifique sua conexão com a internet
- Reinicie o navegador

## 📞 Suporte

Para dúvidas ou problemas técnicos:
- Entre em contato com o administrador do sistema
- Ou com o departamento de TI da ENGERADIOS

## ✅ Checklist de Vistoria

Antes de concluir uma vistoria, verifique:

- [ ] Cliente selecionado corretamente
- [ ] Laudo completo e detalhado
- [ ] Orçamento de adequação preenchido
- [ ] Fotos relevantes adicionadas
- [ ] Áudio gravado (se necessário)
- [ ] Todas as informações revisadas
- [ ] Conexão com internet estável

## 🎯 Boas Práticas

### Para Administradores

1. Cadastre supervisores com e-mails corporativos
2. Defina senhas fortes iniciais
3. Revise vistorias regularmente
4. Faça backup do banco de dados semanalmente
5. Monitore o espaço em disco (fotos e áudios)

### Para Supervisores

1. Faça login no início do dia
2. Preencha relatórios logo após a vistoria
3. Tire fotos claras e bem iluminadas
4. Seja detalhista no laudo
5. Especifique quantidades no orçamento
6. Conclua a vistoria antes de sair do local
7. Verifique se o e-mail foi enviado
